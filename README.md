#  Sistema de Gestión de Jornadas

##  Descripción

El **Sistema de Gestión de Jornadas** es una aplicación web diseñada para registrar, controlar y supervisar las horas de trabajo de los empleados dentro de distintos proyectos.

El sistema permite organizar el trabajo mediante una estructura jerárquica de **proyectos, entregables, fases y tareas**, facilitando el seguimiento del progreso y la toma de decisiones basada en datos.

---

## Objetivo

Desarrollar una solución tecnológica que permita mejorar la organización del trabajo, optimizar el uso del tiempo laboral y aumentar la productividad dentro de una organización.

---

## Funcionalidades principales

* Registro e inicio de sesión de usuarios
* Gestión de proyectos
* Creación de entregables, fases y tareas
* Asignación de tareas a trabajadores
* Registro de horas trabajadas
* Registro de avances y comentarios
* Visualización del progreso de proyectos
* Generación de reportes

---

## Arquitectura del sistema

El sistema está basado en una arquitectura **cliente-servidor**, compuesta por:

* **Frontend:** Interfaz web para interacción con el usuario
* **Backend:** Lógica de negocio y gestión de procesos
* **Base de Datos:** Almacenamiento estructurado de la información

---

## Tecnologías utilizadas

*  **React** – Desarrollo del frontend
*  **Visual Studio Code** – Entorno de desarrollo
*  **Supabase** – Base de datos y backend
*  **GitHub** – Control de versiones

---

##  Estructura del sistema

```text
PROYECTO
 └── ENTREGABLE
      └── FASE
           └── TAREA
                └── REGISTRO DE HORAS
```

---

##  Roles del sistema

###  Administrador

* Crear y gestionar proyectos
* Definir entregables, fases y tareas
* Asignar tareas a trabajadores
* Supervisar el avance

###  Trabajador

* Visualizar tareas asignadas
* Registrar horas trabajadas
* Indicar avance y comentarios

##  Beneficios

* ✔ Permite control preciso de las horas trabajadas
